# Contributing

Thanks for your interest in fluxe.

## Developer Certificate of Origin (DCO) + license grant

By submitting a contribution (pull request, patch, or any code) you agree that:

1. You wrote the contribution or have the right to submit it (DCO 1.1).
2. Your contribution is licensed under **Apache-2.0**, the project license.
3. You grant the project author (**nmvuong92**) a **perpetual, irrevocable,
   non-exclusive right to relicense your contribution** — including under
   commercial or proprietary terms — without further consent or compensation.

This keeps copyright clean and lets the project be offered under separate
commercial terms in the future. Sign-off each commit:

```
git commit -s    # adds: Signed-off-by: Your Name <you@example.com>
```

Contributions without sign-off may be declined.

## File headers

Every source file carries:

```
// Copyright (c) <year> nmvuong92
// SPDX-License-Identifier: Apache-2.0
```

Keep it on new files.
