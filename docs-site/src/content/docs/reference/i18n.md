## Định nghĩa

Đa ngôn ngữ trong fluxe theo mô hình **Resolved Shell**: server **giải locale** từ cookie
`locale` (hoặc `Accept-Language`), bơm `t()` + `locale` vào **loader** và `<html lang>`. Vì dịch
ở loader (server), nội dung đa ngôn ngữ **chạy cả trên cell static (0 JS)**.

## Khai báo catalog

```ts
// app/i18n.ts
import { createI18n } from "@nmvuong92/fluxe";

export const i18n = createI18n({
  defaultLocale: "vi",
  catalogs: {
    vi: { "greet.hello": "Xin chào, {name}!" },
    en: { "greet.hello": "Hello, {name}!" },
  },
});
```

Truyền vào server:

```ts
import { makeServer } from "@nmvuong92/fluxe";
import { i18n } from "./app/i18n";
makeServer(manifest, cells, layouts, { i18n }).listen(5180);
```

## Dùng trong cell (dịch ở loader)

`loader` nhận `t` (bound theo locale) + `locale`:

```tsx
// app/cells/greet/index.tsx
export default defineCell<{}, { greeting: string; locale: string }>({
  id: "greet",
  route: "/greet",
  async loader({ t, locale }) {
    return { greeting: t!("greet.hello", { name: "fluxe" }), locale: locale! };
  },
  view: ({ data }) => <h1>{data.greeting}</h1>,
});
```

## Đổi ngôn ngữ

`<LocaleSwitch>` render link `?locale=xx` — server set cookie + redirect (chạy cả trên trang
static, không cần JS):

```tsx
import { LocaleSwitch } from "@nmvuong92/fluxe/react";
// trong layout, ctx do engine bơm vào: { locale, t, theme, path }
<LocaleSwitch locales={["vi", "en"]} current={ctx.locale} />
```

→ `/greet?locale=en` → cookie `locale=en` + redirect → render tiếng Anh, `<html lang="en">`.

## API

```ts
createI18n({ defaultLocale, catalogs }): I18n
translate(i18n, locale, key, vars?): string     // tra + fallback default + interpolate {var}
makeT(i18n, locale): (key, vars?) => string      // t() bound locale (engine đưa vào loader)
resolveLocale(i18n, { cookie?, acceptLanguage? }): string
```

## Lưu ý

- Quy tắc chọn locale: **cookie hợp lệ > Accept-Language (tag rồi base) > defaultLocale**.
- Thiếu key ở locale → fallback `defaultLocale` → fallback chính `key`.
- Dịch ở **loader** (server) là cách đơn giản nhất — view chỉ render data, không cần i18n phía
  client. Đổi locale = đổi cookie → loader chạy lại → data mới.
- Locale nằm trong **etag render-cache** của cell static → đổi ngôn ngữ tự bust cache, không trả
  HTML sai ngôn ngữ. Xem [Render cache](/guides/static-cache/).
