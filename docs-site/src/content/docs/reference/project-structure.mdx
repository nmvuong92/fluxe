Bạn chỉ làm việc trong **`app/`**, tách **`app/backend`** (API/data) và **`app/frontend`** (cells/UI),
tổ chức theo **feature-module**. Engine là package **`@nmvuong92/fluxe`** (cài qua npm). Không monorepo
tool — chỉ folder + alias `@backend/*`, `@frontend/*`.

<FileTree>
- app/ **CODE CỦA BẠN**
  - backend/ **API + data + nghiệp vụ**
    - db.ts driver data (memory/sqlite/postgres) — `makeDb()`
    - env.ts · contract.ts (spread module) · app.ts (`createApp`)
    - server.ts entry Fastify/Express mount fluxe
    - modules/ **1 feature = 1 module** (đọc api→domain = pipeline)
      - todos/
        - todos.module.ts **entry** (`defineModule` — mở là hiểu cả module)
        - api/ contract.ts · resolver.ts (biên ngoài)
        - domain/ service.ts · data.ts (nghiệp vụ)
    - tests/ **vùng test riêng** (unit/ e2e/ helpers/ fixtures/)
  - frontend/ **cells + UI**
    - features/ **1 feature = cell(s)**
      - home/
        - home.cell.tsx route + loader (server)
        - home.view.tsx giao diện thuần (client)
      - greet/ · todos/ …
    - layouts/ · components/ · i18n.ts · api.ts · profiles.ts
    - registry.ts · views.ts **sinh bởi fx sync** (gitignore)
- node_modules/@nmvuong92/fluxe **ENGINE** (import, không sửa)
- .fluxe/ **artifact**: resolution.json
- dist/ client.js đã bundle
</FileTree>

## Các file `app/` bạn sửa

| File | Vai trò | Tạo bởi |
|------|---------|---------|
| `app/frontend/features/<f>/<name>.view.tsx` | Giao diện thuần (component + kiểu data) | `fx new` |
| `app/frontend/features/<f>/<name>.cell.tsx` | Cell: route + `loader`, gắn view | `fx new` |
| `app/frontend/layouts/` · `components/` | Layout · UI dùng chung | `fx init` |
| `app/backend/modules/<x>/*.plugin.ts` | Feature-module = local plugin | bạn |
| `app/backend/server.ts` | Entry — Express/Fastify, mount fluxe | `fx init` |
| `app/backend/db.ts` | **Tầng data**: driver memory/sqlite/postgres | `fx init` |
| `app/backend/app.ts` | `makeApp()` = `createApp({ plugins })` | `fx init` |
| `app/frontend/registry.ts` · `views.ts` | Registry cell/view | **`fx sync` sinh — đừng sửa** |

## Import từ engine

```ts
import { defineCell, definePlugin, createApp, f } from "@nmvuong92/fluxe";
import { createHooks, Link } from "@nmvuong92/fluxe/react";
import { createClient } from "@nmvuong92/fluxe/client";
```

## Artifact sinh tự động (không sửa tay)

| Đường dẫn | Sinh bởi | Là gì |
|-----------|----------|-------|
| `app/frontend/registry.ts` | `fx sync` | registry cells (server) |
| `app/frontend/views.ts` | `fx sync` | registry views (client) |
| `.fluxe/resolution.json` | `fx resolve` | manifest: mỗi cell giải ra sao |
| `dist/client.js` | build | bundle island |

<Aside title="Quy tắc một câu">
`app/backend` = API/data theo module; `app/frontend` = cells theo feature. Engine
(`@nmvuong92/fluxe`) là dependency — import, đừng sửa. `registry.ts`/`views.ts`, `.fluxe/`, `dist/` sinh ra.
</Aside>
